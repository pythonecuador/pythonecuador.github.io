    Actualmente esta sección no está implementada en el sitio, pero ya que estás interesada/o en esta sección creemos que podrías darnos una mano. Debajo se encuentra una breve descripción de lo que deberías implementar.

    ¿Qué debe ir en esta página?

    Esta página debe tener una lista de los canales de comunicación de Python Ecuador. Y dentro de este directorio (pages/comunidades/) debe existir una página por cada ciudad. Sólo se necesaría crear páginas con un contenido parecido a esta página, incentivando a las personas que sean de esa ciudad a colaborar o a organizar un grupo en su ciudad.

    Las página de cada ciudad deben contener información general acerca de la comunidad, como: dónde se está organizando eventos, o con quién se pueden comunicar para organizar uno esa ciudad.

    ¡Quiero ayudar! ¿Cómo lo hago?

    ¡Muchas gracias por el interés en ayudar! El desarrollo de nuestro sitio se lleva a cabo en GitHub sobre este repositorio pythonecuador/pythonecuador.github.io.

    El sitio es un sitio estático construído con html, css y javascript, y alojado en GitHub Pages. Pero no necesitas saber nada de esto para empezar a colaborar, cualquier tipo de ayuda es muy importatante. Desde corregir una falta ortográfica, agregar documentación, revisiones, etc.

    Hemos redactado una guía con el paso a paso para que puedas iniciar a colaborar, la puedes leer aquí. Si te quedas atascada/o en un paso no dudes en preguntar en nuestro canal de telegram o en algún otro de nuestros canales de comunicación.
