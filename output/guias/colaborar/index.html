    ¡Quiero ayudar! ¿Cómo lo hago?

    ¡Muchas gracias por el interés en ayudar! El desarrollo de nuestro sitio se lleva a cabo en GitHub sobre este repositorio pythonecuador/pythonecuador.github.io.

    El sitio es un sitio estático construído con html, css y javascript, y alojado en GitHub Pages. Pero no necesitas saber nada de esto para empezar a colaborar, cualquier tipo de ayuda es muy importatante. Desde corregir una falta ortográfica, agregar documentación, revisiones, etc.

    Hemos redactado una guía con el paso a paso para que puedas iniciar a colaborar, la puedes leer aquí. Si te quedas atascada/o en un paso no dudes en preguntar en nuestro canal de telegram o en algún otro de nuestros canales de comunicación.
